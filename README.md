# 佳宜幼小衔接 - 综合启蒙学习应用

一款专为6岁幼小衔接儿童设计的综合启蒙Web应用，整合**数学启蒙**、**拼音学习**和**英语启蒙**三大核心模块，界面童趣简洁、操作极简，支持全端适配（手机/平板/电脑），可打包为桌面应用。

## 功能特色

### 🔢 数学乐园

#### 基础运算模块
- **10以内加减法**：可选择加法、减法或混合运算，包含填空题型
- **20以内加减法**：含进退位运算、复合运算，配合可视化辅助
- **拆数运算**：数字分解与组合练习（如 5 = 1+4 = 2+3）

#### 数感培养模块
- **相邻数练习**：找出数字的前后相邻数，多种题型变化
- **比大小**：数字比较练习，选择 > < = 符号
- **数列规律**：顺数、倒数、跳数等规律填空
- **生活应用题**：贴近生活的数学应用场景（购物、分配、比较等）

### 🔤 拼音天地

#### 拼音学习模块
- **韵母学习**：单韵母(a o e i u ü)、复韵母(ai ei ui等)、前后鼻韵母
- **声母学习**：23个声母分类学习，唇音、舌尖音、平翘舌音区分
- **整体认读**：16个整体认读音节专项学习
- **声调学习**：四声标注、声调练习

#### 拼读训练模块
- **两拼法**：声母+韵母拼读（如 b+a→ba）
- **三拼法**：声母+介母+韵母拼读（如 j+i+a→jia）
- **拼读动画**：声母与韵母"碰撞"合成音节的动态动画
- **颜色区分**：声母蓝色、韵母红色、介母绿色
- **声调选择**：支持四声切换，准确发音

#### 拼音练习
- **拼音认读**：看拼音选汉字/图片
- **汉字注音**：看汉字选拼音
- **平翘舌辨析**：z/zh、c/ch、s/sh 区分练习

#### 拼音工具
- **拼音字母表**：完整可交互字母表，点击播放发音
- **拼音收藏夹**：收藏难记的拼音，支持重复练习

### 🔤 英语启蒙

#### 单词学习
- **单词闪卡**：200+常用英语单词，18个主题分类
- **单词发音**：标准美式发音（有道词典TTS）
- **音标展示**：每个单词配有国际音标
- **中文释义**：简单易懂的中文解释

#### 常用句型
- **10大场景**：问候、介绍、请求、喜好、天气等
- **例句学习**：每个句型配有多个实用例句
- **句子发音**：完整句子朗读

#### 单词练习
- **看词选义**：看英文单词选中文意思
- **看义选词**：看中文意思选英文单词
- **听音选词**：听发音选择正确单词

### 🎮 趣味互动模块
- **闯关模式**：入门关/进阶关/挑战关三个难度等级，1-3星评价
- **错题本**：自动收集错题，支持重复练习和一键清除
- **学习记录**：今日做题数、正确率、各科目学习进度统计

### ⚙️ 通用设置
- 字体大小调节（小/中/大三档）
- 音效开关
- 拼音发音速度调节（慢速/正常/快速）
- 数据重置
- 关于页面

## 技术特点

### 语音系统
- **有道词典TTS**：用于短词语发音（单词、拼音），发音清晰标准
- **Web Speech API**：用于长句子朗读（数学应用题、英语句子）
- **智能切换**：根据文本长度自动选择最佳语音方案
- **请求优化**：缓存机制 + 限流保护，避免API过载

### 界面设计
- **马卡龙配色**：柔和护眼的糖果色系
- **扁平化图标**：统一的SVG矢量图标
- **固定导航**：顶部返回栏固定，方便操作
- **响应式布局**：完美适配手机、平板、电脑

### 兼容性
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 支持微信内置浏览器
- 支持移动端触摸操作
- 可打包为桌面应用（Tauri）

## 技术栈

- **框架**: React 18 (函数组件 + Hooks)
- **UI组件库**: Mantine v7
- **样式**: Tailwind CSS v3
- **构建工具**: Vite 5
- **路由**: React Router v6
- **语音**: 有道词典TTS + Web Speech API
- **存储**: localStorage
- **类型**: TypeScript

## 操作指引

### 首页
- 显示今日学习统计和综合正确率
- 点击"数学乐园"进入数学模块
- 点击"拼音天地"进入拼音模块
- 点击"英语启蒙"进入英语模块
- 快捷入口直达各个功能

### 数学练习
1. 选择想练习的题型（加法、减法、应用题等）
2. 设置题目数量（5-20题）
3. 点击"开始练习"
4. 点击正确答案或输入数字答题
5. 完成后查看得星情况

### 拼音学习
1. 选择学习内容（韵母、声母、整体认读）
2. 点击任意拼音卡片听发音
3. 进入拼读训练学习两拼法、三拼法
4. 使用拼音表查看完整字母表
5. 进入拼音练习检验学习效果

### 英语学习
1. 进入单词闪卡浏览单词
2. 按分类学习不同主题的单词
3. 查看常用句型学习日常表达
4. 进入单词练习检验学习效果

### 家长辅助指南
- 可在设置中调整字体大小适应孩子视力
- 开启/关闭声音效果
- 调整拼音发音速度（慢速更适合初学）
- 查看学习统计了解孩子学习情况
- 使用错题本帮助孩子复习薄弱环节

## 桌面应用打包 (Tauri)

### 环境准备

1. 安装 Rust: https://www.rust-lang.org/tools/install

2. 安装 Tauri CLI:
```bash
npm install -g @tauri-apps/cli
```

### 初始化 Tauri

```bash
npm install @tauri-apps/api
npx tauri init
```

配置 `src-tauri/tauri.conf.json`:

```json
{
  "build": {
    "distDir": "../dist",
    "devPath": "http://localhost:5173",
    "beforeBuildCommand": "npm run build",
    "beforeDevCommand": "npm run dev"
  },
  "package": {
    "productName": "佳宜幼小衔接",
    "version": "3.0.0"
  },
  "tauri": {
    "bundle": {
      "active": true,
      "identifier": "com.joye.preschool",
      "icon": ["icons/icon.png"]
    },
    "windows": [
      {
        "title": "佳宜幼小衔接",
        "width": 1024,
        "height": 768,
        "resizable": true,
        "fullscreen": false
      }
    ]
  }
}
```

### 打包

```bash
# 开发模式
npx tauri dev

# 构建发布版
npx tauri build
```
